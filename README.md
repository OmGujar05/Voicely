🎙️ Voicely - Voice Based Email Application

Voicely is an accessibility-focused voice-based email application designed to empower visually impaired users to send, receive, and manage emails through voice interactions. By leveraging Speech-to-Text (STT) and Text-to-Speech (TTS) technologies, Voicely provides a seamless hands-free email experience.

✨ Features

• 🎤 Voice-based email composition
• 🔊 Text-to-Speech for reading emails aloud
• 🗣️ Speech-to-Text for voice input
• 📧 Send and receive emails using voice commands
• ♿ Accessibility-focused user interface
• 📱 Cross-platform Flutter application

🛠️ Tech Stack

• Flutter
• Dart
• Node.js
• Express.js
• REST APIs
• Speech-to-Text (STT)
• Text-to-Speech (TTS)
• mysql (database)

🚀 Getting Started Prerequisites

• Flutter SDK
• Android Studio / VS Code
• Android Emulator or Physical Device

🚀 Installation

1. Clone the repository
git clone https://github.com/your-username/voicely.git
cd voicely

2. Setup the Backend
cd backend
npm install
node index.js

The backend server will start on the configured port.

3. Setup the Frontend
Open a new terminal:
cd frontend
flutter pub get
flutter run

4. Ensure that the backend server is running before launching the Flutter application.
